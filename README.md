# Real Estate Investment Advisor (Advanced)

This repository contains an advanced implementation of the Real Estate Investment Advisor project.
It includes robust preprocessing, XGBoost training (script), MLflow logging (script), and a Streamlit app.

**How to use**
1. Create virtual env and install: pip install -r requirements.txt
2. (Optional) Preprocess: python src/preprocess.py
3. (Optional) Train models: python src/train_xgboost.py
4. Run Streamlit: streamlit run streamlit_app/app.py

If you used the uploaded dataset, it is saved in data/india_housing_prices.csv
